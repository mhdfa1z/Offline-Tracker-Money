# Offline Tracker Money

Offline Tracker Money adalah aplikasi pencatat keuangan pribadi (personal finance tracker) yang dirancang untuk membantu pengguna mengelola pemasukan dan pengeluaran dengan mudah tanpa memerlukan koneksi internet. Aplikasi ini berfokus pada kesederhanaan, kecepatan, dan privasi, sehingga seluruh data disimpan secara lokal di perangkat pengguna.

Dengan Offline Tracker Money, pengguna dapat mencatat setiap transaksi keuangan, mengelompokkan transaksi berdasarkan kategori, memantau saldo secara real-time, serta melihat riwayat transaksi kapan saja. Karena bekerja sepenuhnya secara offline, aplikasi tetap dapat digunakan di mana pun tanpa bergantung pada layanan cloud atau jaringan internet.

## Tujuan Proyek

Proyek ini dibuat untuk menyediakan solusi pencatatan keuangan yang ringan, cepat, dan mudah digunakan oleh siapa saja. Selain membantu mengelola keuangan sehari-hari, aplikasi ini juga dapat menjadi contoh implementasi pengembangan aplikasi dengan penyimpanan data lokal dan antarmuka yang sederhana.

## Fitur Utama

* Menambahkan transaksi pemasukan.
* Menambahkan transaksi pengeluaran.
* Mengelola kategori transaksi.
* Menampilkan saldo secara otomatis berdasarkan seluruh transaksi.
* Riwayat transaksi yang lengkap.
* Pencarian dan penyaringan transaksi.
* Statistik sederhana pengeluaran dan pemasukan.
* Edit dan hapus transaksi.
* Penyimpanan data secara lokal.
* Berjalan sepenuhnya tanpa koneksi internet.
* Antarmuka yang sederhana, responsif, dan mudah digunakan.

## Keunggulan

* **100% Offline** — Tidak memerlukan koneksi internet.
* **Privasi Terjaga** — Data tidak dikirim ke server mana pun.
* **Ringan dan Cepat** — Performa optimal karena seluruh data diproses secara lokal.
* **Mudah Digunakan** — Desain antarmuka yang sederhana sehingga mudah dipahami oleh semua pengguna.
* **Aman** — Data tetap berada di perangkat pengguna.

## Manfaat

Offline Tracker Money membantu pengguna untuk:

* Mengontrol pengeluaran harian.
* Mencatat seluruh pemasukan.
* Memantau kondisi keuangan secara berkala.
* Mengetahui kebiasaan pengeluaran.
* Membantu menyusun anggaran pribadi.
* Mengurangi risiko lupa mencatat transaksi.

## Target Pengguna

Aplikasi ini cocok digunakan oleh:

* Pelajar dan mahasiswa.
* Karyawan.
* Freelancer.
* Pemilik usaha kecil (UMKM).
* Siapa saja yang ingin mengelola keuangan pribadi secara sederhana.

## Pengembangan Selanjutnya

Beberapa fitur yang direncanakan untuk versi berikutnya antara lain:

* Backup dan Restore data.
* Ekspor ke PDF dan Excel.
* Grafik keuangan yang lebih lengkap.
* Pengingat pencatatan transaksi.
* Multi akun/dompet.
* Multi mata uang.
* Tema gelap (Dark Mode).
* Sinkronisasi opsional melalui cloud.
* Proteksi aplikasi menggunakan PIN atau biometrik.

## Lisensi

Proyek ini dikembangkan sebagai aplikasi open source dan dapat digunakan, dipelajari, serta dikembangkan lebih lanjut sesuai dengan ketentuan lisensi yang digunakan pada repository ini.
